# Dessert Recipe App Walkthrough

## How to Test

1.  **Open the App**:
    - Go to [http://localhost:5173](http://localhost:5173) (or the port shown in your terminal).

2.  **Test Recipe Generation**:
    - Type a dessert name (e.g., "Chocolate Cake", "Bingsu", "บราวนี่").
    - Click **Search** (or "ค้นหา").
    - **Verify**: You should see ingredients and instructions generated by Gemini.

3.  **Test Validation**:
    - Type something that is NOT a dessert (e.g., "Pad Thai", "Fried Rice").
    - **Verify**: The app should show an error message saying it can only help with desserts.

4.  **Test Favorites**:
    - When a recipe is generated, click the **Heart icon** (Pin) in the top right of the card.
    - Go to the **Favorites** page (click "Favorites" or "รายการโปรด" in the nav bar).
    - **Verify**: The recipe you saved should be listed there.

5.  **Test Language Switcher**:
    - Click the **TH/EN** button in the top right.
    - **Verify**: The interface text changes between Thai and English.
    - **Note**: Recipes generated in Thai mode will be in Thai, and vice versa.

## New Features (v2)
- **Mood Roulette**: Generate recipes based on your mood (Heartbroken, Tired, Celebrate, Surprise Me).
- **Portion Calculator**: Adjust serving sizes with +/- buttons.
- **Cooking Mode**: Hands-free view with large text and step-by-step navigation.
- **Allergy Filter**: Exclude ingredients like Nuts, Dairy, Gluten, or Eggs.
- **Enhanced UI**:
    - Collapsible recipe cards to save space.
    - Denser grid layout for favorites.
    - Custom background pattern.
    - "Popular Desserts" quick access.
